\section*{Introduction to Using
Functions}\label{introduction-to-using-functions}}

In this section we shall discuss some of the details on how to use
functions from the Standard Library. We have already come across a few
functions, namely \texttt{print()} and \texttt{type()}. We shall cover
how to define your own functions in a later section.

Python functions essentially take variables or values/objects as
arguments, perform a task and them return values/objects. As we have
seen before the syntax of a function call is:

\begin{verbatim}
function_name(argument, argument, ...)
\end{verbatim}

Note that some functions return a \texttt{None} type when a return value
isn't necessary. For example, the \texttt{print()} function:



    \begin{tcolorbox}[breakable, size=fbox, boxrule=1pt, pad at break*=1mm,colback=cellbackground, colframe=cellborder]
\prompt{In}{incolor}{4}{\boxspacing}
\begin{Verbatim}[commandchars=\\\{\}]
\PY{n}{print\PYZus{}return} \PY{o}{=} \PY{n+nb}{print}\PY{p}{(}\PY{l+s+s1}{\PYZsq{}}\PY{l+s+s1}{print out}\PY{l+s+s1}{\PYZsq{}}\PY{p}{)}

\PY{n+nb}{print}\PY{p}{(}\PY{l+s+s1}{\PYZsq{}}\PY{l+s+s1}{print function return:}\PY{l+s+s1}{\PYZsq{}}\PY{p}{,} \PY{n}{print\PYZus{}return}\PY{p}{)}
\end{Verbatim}
\end{tcolorbox}

    \begin{Verbatim}[commandchars=\\\{\}]
print out
print function return: None
    \end{Verbatim}



    If you want to know what a particular function does or how to use it, a
quick way to find out is to pull up the docstring. In an IPython
environment (such as a Jupyter notebook) this can be done by typing a
\texttt{?} symbol after the function name and pressing enter. For
example:



    \begin{tcolorbox}[breakable, size=fbox, boxrule=1pt, pad at break*=1mm,colback=cellbackground, colframe=cellborder]
\prompt{In}{incolor}{5}{\boxspacing}
\begin{Verbatim}[commandchars=\\\{\}]

\end{Verbatim}
\end{tcolorbox}

Now, let's take a look at the \texttt{print()} docstring itself. The
first line of the docstring shows us the function name; and the name of
the function arguments and their default values (if they have).
Following this is a description of what the function does. Following
this is a list of optional keyword arguments.

Sometimes a docstring will also contain examples on how to use the
function, though none are present in this one.

We will see more examples of how optional keyword arguments work (in
particular in the Chapter discussing Matplotlib\ldots{}), for now let's
use one of them. Let's change the argument \texttt{sep}, which is the
string inserted between the values you put into the \texttt{print()}
function. As we can see, it's default value is a space
\texttt{\textquotesingle{}\ \textquotesingle{}}. As a first example,
let's change the separation to an empty string (no space):

    \hypertarget{the-input-function}{%
\subsection*{\texorpdfstring{The \texttt{input()}
Function}{The input() Function}}\label{the-input-function}}





    Another important function in the Python Standard Library is the
\texttt{input()} function. This function allows us to collect user
inputs from the terminal.

\texttt{input()} takes a string as an argument, this gets printed to the
terminal and the script is halted until the user has entered a string
and pressed enter. This string that the user has entered is returned by
the \texttt{input()} function; and can, therefore, be used in the rest
of the script.

As a first example, consider the following code that asks the user for
their name:



    \begin{tcolorbox}[breakable, size=fbox, boxrule=1pt, pad at break*=1mm,colback=cellbackground, colframe=cellborder]
\prompt{In}{incolor}{5}{\boxspacing}
\begin{Verbatim}[commandchars=\\\{\}]
\PY{n}{user\PYZus{}name} \PY{o}{=} \PY{n+nb}{input}\PY{p}{(}\PY{l+s+s1}{\PYZsq{}}\PY{l+s+s1}{What is your name?}\PY{l+s+s1}{\PYZsq{}}\PY{p}{)}

\PY{n+nb}{print}\PY{p}{(}\PY{l+s+s1}{\PYZsq{}}\PY{l+s+s1}{Hello}\PY{l+s+s1}{\PYZsq{}}\PY{p}{,} \PY{n}{user\PYZus{}name}\PY{p}{,} \PY{l+s+s1}{\PYZsq{}}\PY{l+s+s1}{, nice to meet you!}\PY{l+s+s1}{\PYZsq{}}\PY{p}{)}
\end{Verbatim}
\end{tcolorbox}

    \begin{Verbatim}[commandchars=\\\{\}]
What is your name? Mayhew
    \end{Verbatim}

    \begin{Verbatim}[commandchars=\\\{\}]
Hello Mayhew , nice to meet you!
    \end{Verbatim}



    Remember that the program will wait for your input before it continues.
If you are using a Jupyter Notebook, this means that other cells from
the notebook will not run until the code has been fully executed or
terminated.

Now, something that is important to note is that the return value from
input is a string, even when a number is typed into the terminal:



    \begin{tcolorbox}[breakable, size=fbox, boxrule=1pt, pad at break*=1mm,colback=cellbackground, colframe=cellborder]
\prompt{In}{incolor}{7}{\boxspacing}
\begin{Verbatim}[commandchars=\\\{\}]
\PY{n}{user\PYZus{}number} \PY{o}{=} \PY{n+nb}{input}\PY{p}{(}\PY{l+s+s1}{\PYZsq{}}\PY{l+s+s1}{Enter a number: }\PY{l+s+s1}{\PYZsq{}}\PY{p}{)}

\PY{n+nb}{print}\PY{p}{(}\PY{n}{user\PYZus{}number}\PY{p}{,} \PY{n+nb}{type}\PY{p}{(}\PY{n}{user\PYZus{}number}\PY{p}{)}\PY{p}{)}
\end{Verbatim}
\end{tcolorbox}

    \begin{Verbatim}[commandchars=\\\{\}]

    \end{Verbatim}

    \begin{Verbatim}[commandchars=\\\{\}]

    \end{Verbatim}
